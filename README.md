# Smart Task Manager (AI Powered)

## ✨ Features
- Create / Edit / Delete tasks
- Suggest 3-5 subtasks using Gemini AI
- Responsive design (mobile & desktop)
- Gemini API integration via Next.js API route

## 🧠 Technologies Used
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Google Gemini API
